В соответствующих разделах документа «Курсовые и выпускные работы» доступны инструкции по подготовке отчетов о курсовой работе или выпускных работ и презентаций.

Для упрощения работы студента разработаны шаблоны отчета в макропакете LaTeX и презентации в его классе Beamer. Использование шаблонов не требует глубокого знания инструментов TeX, LaTeX, Beamer, достаточно уметь отличать их конструкции от содержательного текста. В случае затруднений можно воспользоваться описанием языка LaTeX, документацией пакета Beamer на английском языке, некоторыми полезными примерами, обратиться к руководителю или к дежурному по кафедре.

Шаблоны можно выгрузить в рабочий каталог в архивах формата ZIP:

  1. Шаблоны отчетов:
    • промежуточный;
    • выпускная квалификационная работа бакалавра;
    • магистерская диссертация.
  2. Шаблоны презентаций:
    • промежуточная:
      • для студентов кафедры ИМО
      • для студентов кафедры ПМиК
    • выпускная квалификационная работа бакалавра:
      • для студентов кафедры ИМО
      • для студентов кафедры ПМиК
    • магистерская диссертация:
      • для студентов кафедры ИМО
      • для студентов кафедры ПМиК

Также, файлы шаблонов доступны для просмотра и скачивания в соответсвующих каталогах в файловой системе кафедры.

После распаковки любого из архивов шаблонов ЭП будут доступны файл "slidex.tex", содержащий структуру и текст презентации, файл темы оформления презентации (с расширением ".sty") и каталог "images" для иллюстраций (основные форматы JPEG, PNG, PDF).

Архив с шаблоном промежуточного отчета содержит файл "report.tex" и каталог "images" для иллюстраций.

Архивы с шаблонами выпускной квалификационной работы бакалавра и магистерской диссертации содержат несколько файлов с расширением ".tex" и каталог "images" для иллюстраций.
